Как запускать:
	Делаем виртуальное окружение, для этого:
	1) Скачиваем и устанавливаем python версии не ниже 3.11 (при условии, что на компьютере его нет)
	2) Открываем терминал (в случае с виндой, сойдет cmd)
	3) В терминале переходим в каталог с проектом (папка выше данного проекта)
	4) В каталоге с проектом в терминале выполняем команду python -m venv Practice\venv
		! Если на компьютере не винда, а линукс, то вместо python пишем python3
		! Путь должен выглядеть примерно так D:\Work>python -m venv Practice\venv
	5) Переходим в рабочую дирректорию (папка Practice)
		5.1) Переходим в папку \venv\ далее в \scripts\, от туда выполняем файл activate
			! Должно выглядеть примерно так: D:\Work\Practice\venv\Scripts>activate
			! После активации должно получится так: (venv) D:\Work\Practice\venv\Scripts>
	6) Поднимаемся обратно в рабочую дирректорию (папка Practice) и запускаем команды по очереди
		! pip install django (собственно установит сам фреймворк)
		! pip install pillow (необходима для взаимодействия с медиа-данными)
		! pip install pytils (для построения слага, относительно символов на кириллице)
	7) В рабочей дирректории запускаем команду
		! python manage.py runserver
	8) Переходим по предложенному в терминале адресу
		! По умолчанию http://127.0.0.1:8000
